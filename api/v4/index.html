ABF Browser API (Version 4)

Goals of this structure include:
  • improved record keeping
  • promote daily e-phys folders
  • simplify how ABFs are grouped for large projects
  • improved searching
Project Folder
  • A project is a collection of experiments
  • Projects are in X:/Projects/ProjectName
  • Jason should be involved in the naming of projects
API Requests
  • project/?path=X:\Projects\Aging-eCB
Experiment Folders
  • Large projects consist of many experiments
  • Experiment folders live in ProjectName/Experiments/
  • An experiment folder may contain many daily ABF folders
  • Experiment folders may be created for 2P or IHC data (they are not just for daily e-phys folders)
API Requests
  • experiment/?path=X:\Projects\Aging-eCB\abfs\SA4503 time course NAc Core
  • experiment/?path=X:/Projects/Sigma1R/experiments/METH intrinsic properties
  • experiment/?path=X:\Projects\Aging-eCB\experiments\DSI in CA1
  • experiment/?path=X:\Projects\Aging-eCB\experiments\DSI in PFC
Daily ABF Folders
  • An "abf day" folder is a special daily data folder just e-phys experiments.
  • It contains abfday.json which describes experiment details for that day.
  • abfday.json also contains notes about cells (color, comment, and group).
API Requests
  • abfday/?path=X:\Projects\Sigma1R\experiments\METH intrinsic properties\2021-12-03
  • abfday/?path=X:\Projects\Aging-eCB\experiments\DSI in CA1\2022-03-04
  • abfday/?path=X:\Projects\Aging-eCB\experiments\DSI in CA1\2022-03-07
  • abfday/?path=X:\Projects\Aging-eCB\experiments\DSI in CA1\2022-03-08
  • abfday/?path=X:\Projects\Aging-eCB\experiments\DSI in CA1\2022-03-10
